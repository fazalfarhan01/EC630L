# Common Drain Amplifier - Source Follower
### Given
- `Id` = 0.35 mA
- at `Vb` = 1.5V
- `Vout` = 1.82V
- `(W/L)1` = 6/6 = 1
- `(W/L)2` = 1.85 ~= 2
- `W2` = 4
- `l2` = 2

### Observations
- Gain: -1.99 db
- Bandwidth = 4.78GHz

# Common Gate Amplifier
### Given
- Designed Gain `Av` = 10
    - `Av = gm1 * Rd`
    - `Rd = 1/gm2`
    - `Av = gm1/gm2`
    - `Av^2 = 2*(W/L)1 / (W/L)2`
    - Assuming `(W/L) = 6/1`
    - `(W/L)2 = 3/16`

### Observations
- Transient Analysis
    - Gain: 9.5
- AC Analysis
    - Gain = `10.59` = `20.5 db`
    - Bandwidth = `106.6 MHz`
